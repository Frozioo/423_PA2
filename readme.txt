Provided in this folder is the Collatz.c file and the readme file.

In order for the user to run the Collatz.c file, the user should compile the file by doing:

gcc Collatz.c -o b

Make sure you are in the correct directory where the file is located in!

After compiling, the user should then input this line into the terminal:

./b integer

The integer variable can be any POSITIVE integer. For example, it can be 8. If it were -2, it will give you an error.

If the user were to use the example provided, which is 8. The user should expect this output:

8, 4, 2, 1

Another example is 3, if the user were to input 3, the expected output is:

3, 10, 5, 16, 8, 4, 2, 1
